# PlayLook

PlayLook is a playback viewer and play/pause button, using SystemMediaTransportControls.
It is a simple and easy-to-use library that allows you to control media playback in your application.

## Japanese

PlayLookは、SystemMediaTransportControlsを使用した再生ビューアと再生/一時停止ボタンです。

### Features

- 現在再生中のメディアの情報をSMTCから取得して表示する
- マテリアルデザインで見やすく
- 大きな再生・一時停止ボタン
- かさばらないようタスクトレイに格納できる
- ダークモードに対応
- メディア情報をWebサーバーに同期(オプション)

## Requirements (For Development)

- Windows 11
- .NET 8.0
- C# 11.0
- Visual Studio 2022
- Windows App SDK 1.4.0
- WinRT 3.1
